# GameHub-Solidario
GameHub Solidario – Proyecto académico de frontend desarrollado para la materia Taller de Lenguajes de Marcado y Tecnologías Web (2° cuatrimestre 2025). Sitio web responsive para una comunidad que organiza torneos de videojuegos con fines solidarios, incluyendo panel de administración, sistema de login y formularios dinámicos de colaboración.
